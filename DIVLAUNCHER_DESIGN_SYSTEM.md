# Дизайн-система DivLauncher

Этот документ описывает визуальный стиль, компоненты и анимации, используемые в экосистеме DivLauncher. Он предназначен для сохранения единого стиля при разработке сайта, профиля пользователя и новых интерфейсов лаунчера.

## 1. Типографика
- **Основной шрифт**: `Montserrat`, sans-serif.
- У шрифта используется широкий спектр начертаний: от `600` (SemiBold) для обычного текста до `800` (ExtraBold) и `900` (Black) для заголовков.
- **Особенности**: часто используется `text-transform: uppercase` и увеличенный `letter-spacing` (например, `1.6px`) для подзаголовков и меток (labels).

## 2. Цветовая палитра
### Основной фон и панели
- **Фон приложения**: Тёмный градиент поверх размытого изображения или видео.
  `linear-gradient(rgba(20, 20, 20, 0.7), rgba(20, 20, 20, 0.9))`
- **Панели (Glassmorphism)**: Используется полупрозрачный тёмный фон с размытием заднего плана.
  `background: rgba(10, 10, 16, 0.5)`
  `backdrop-filter: blur(24px)`
  `border: 1px solid rgba(255, 255, 255, 0.07)`

### Текст
- **Основной текст**: `#ffffff` (Белый)
- **Второстепенный текст**: `#e4e4e7` (Светло-серый), `#a1a1aa` (Серый), `#71717a` (Тёмно-серый)
- **Лейблы (Метки)**: `#52525b`

### Акцентные цвета (Используются для кнопок, иконок и статусов)
Каждый раздел или состояние имеет свой акцентный цвет. Часто используется вместе с полупрозрачным фоном того же цвета.
- **Успех / Игра / Основной акцент (Emerald)**: `#10b981` (Hover: `#34d399`)
- **Профиль (Purple)**: `#a78bfa` / `#7c3aed`
- **Сеть / Информация (Blue)**: `#3b82f6` / `#60a5fa`
- **Ошибки / Сброс / Java (Red)**: `#ef4444` / `#f87171`
- **Настройки / Ожидание (Amber)**: `#f59e0b` / `#fbbf24`

## 3. Элементы UI

### Кнопки
- **Акцентные кнопки**: Имеют градиентный фон, бордер и эффекты при наведении.
  Пример (Изумрудная кнопка):
  ```css
  background: linear-gradient(135deg, rgba(16,185,129,0.14), rgba(5,150,105,0.09));
  border: 1px solid rgba(16,185,129,0.32);
  color: #34d399;
  border-radius: 11px;
  padding: 9px 14px;
  font-weight: 800;
  ```
- **Состояния**: При `hover` кнопка слегка увеличивается (`scale: 1.03`) и получает тень (`box-shadow: 0 0 14px rgba(..., 0.2)`). При нажатии (`tap`) уменьшается (`scale: 0.96`).

### Поля ввода (Inputs)
- **Стиль**: Глубокие, полупрозрачные поля.
  ```css
  background: rgba(255, 255, 255, 0.05);
  border: 1px solid rgba(255, 255, 255, 0.1);
  border-radius: 13px;
  color: #fff;
  padding: 12px 16px;
  ```
- **Фокус**: При фокусе граница меняет цвет на основной акцентный (`#10b981`).

### Карточки (Section Cards)
- Карточки имеют сильное скругление (`border-radius: 20px`), эффект стекла и тонкий белый бордер (opacity 0.07).
- Внутри карточек часто используются маленькие "бейджики" с иконками, где фон это акцентный цвет с прозрачностью 10% (`18` в HEX), а бордер — 15% (`28` в HEX).

## 4. Анимации и взаимодействие
- Используется библиотека `framer-motion` (в вебе) для плавных переходов.
- **Появление (Mounting)**: Элементы появляются с `opacity: 0` и сдвигом `y: 10` или `y: 16`.
- **Модальные окна**: Эффект пружины (`type: "spring", stiffness: 400, damping: 30`), появление из `scale: 0.88`.
- **Микроинтеракции**:
  - Иконки загрузки плавно вращаются (`rotate: 360`, `linear`).
  - Переключатели (Toggles) имеют пружинную анимацию ползунка.
  - Hover-эффекты применяются ко всем кликабельным элементам.

## 5. Иконки
- **Шрифт иконок**: `FontAwesome 6 Solid` (`fa-solid`).
- Иконки всегда сопровождаются небольшим отступом (gap) от текста (обычно `6px` - `10px`).

---
При создании веб-сайта с личным кабинетом, разработчики должны строго придерживаться этих принципов, чтобы сайт выглядел как естественное продолжение лаунчера.
