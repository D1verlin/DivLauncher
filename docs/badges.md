# 🏷️ Система Бейджей (Badges)

Бейджи — это цветные префиксы перед ником игрока, которые выдаются вручную через **Админ-панель** лаунчера.

---

## Как выдать бейдж игроку

1. Войдите в лаунчер под аккаунтом **администратора**
2. Перейдите в **Админ-панель** (иконка щита в навигации)
3. Найдите нужного игрока в списке (поиск по нику или UUID)
4. Нажмите кнопку **редактирования** (иконка карандаша)
5. В поле **«Бэйдж»** введите название бейджа (например: `VIP`, `Dev`, `Moder`)
6. Нажмите **Сохранить**

Бейдж немедленно появится перед ником игрока во всех местах лаунчера (профиль, страница игроков, шапка).

---

## Как сделать себя/кого-то администратором

1. Откройте файл базы данных напрямую через **SQLite**:
   ```
   backend/users.db
   ```
2. Выполните SQL-запрос:
   ```sql
   UPDATE users SET is_admin = 1 WHERE username = 'ВашНик';
   ```
3. Перезапустите бэкенд лаунчера и перелогиньтесь

   **Либо** — если у вас уже есть аккаунт администратора, используйте Админ-панель для выдачи прав.

## Предустановленные стили бейджей

| Бейдж (ввести в поле) | Цвет | Предназначение |
|---|---|---|
| `ADMIN` / `АДМИН` / `OWNER` | 🔴 Красный | Администратор сервера |
| `DEV` / `DEVELOPER` / `РАЗРАБОТЧИК` | 🔵 Голубой | Разработчик проекта |
| `VIP` / `ВИП` / `GOLD` | 🟡 Золотой | VIP/донатер |
| `PREMIUM` / `PREM` / `ПРЕМИУМ` | 🟢 Зелёный | Премиум-игрок |
| `YOUTUBE` / `YT` / `MEDIA` | 🔴 Красно-оранжевый | Медиа-персона |
| `SPONSOR` / `СПОНСОР` | 🩷 Розово-фиолетовый | Спонсор сервера |
| `HELPER` / `MOD` / `МОДЕРАТОР` | 🟣 Фиолетовый | Модератор |
| *Любое другое значение* | 🌈 Фиолетово-зелёный | Кастомный бейдж |

Стиль определяется в файле `src/ui/utils/badgeHelper.js`.

---

## Как добавить новый кастомный стиль бейджа

Откройте файл `src/ui/utils/badgeHelper.js` и добавьте блок в функцию `getBadgeStyle`:

```js
} else if (b === 'BUILDER' || b === 'СТРОИТЕЛЬ') {
  gradient = 'linear-gradient(135deg, #f97316, #b45309)';
  border = 'rgba(249, 115, 22, 0.35)';
}
```

Значение `b` — это бейдж, приведённый к **верхнему регистру** (`.toUpperCase()`), поэтому проверяйте в верхнем регистре.

## Хранение бейджей в базе данных

Бейдж хранится в колонке `badge` таблицы `users` (тип `TEXT`, допускает `NULL`):

```sql
-- Посмотреть все бейджи
SELECT username, badge FROM users WHERE badge IS NOT NULL;

-- Выдать бейдж напрямую
UPDATE users SET badge = 'VIP' WHERE username = 'PlayerName';

-- Убрать бейдж
UPDATE users SET badge = NULL WHERE username = 'PlayerName';
```

База данных находится по пути: `backend/users.db`
